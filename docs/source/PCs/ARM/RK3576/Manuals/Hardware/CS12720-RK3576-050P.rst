.. |HDMI_des| replace:: 1 x HDMI-D (Micro-HDMI) Out

.. |USB| replace:: 1 x USB 2.0 HOST, 1 x USB 3.0 HOST, 1 x USB Type-C

.. |USB_conflict| replace:: USB 3.0 port and USB-C port cannot be used at the same time

.. |usb_usbc_conflict| replace:: The USB 3.0 type-A host and USB-C **can't be used** at the same time. Before boot into OS, USB-C is enabled for installing OS image; after boot into OS, USB-A is enabled but USB-C is disabled. |br| In Android, these can be configured, USB3.0 type A is enabled by default, but you can switch to using USB-C in the software for debugging; in Linux, these can't be configured.

.. |LAN| replace:: 1 x RJ45, GbE

.. |LAN_des| replace:: 1 x RJ45

.. |npu| replace:: 6 TOPS(Sparsity)@INT8

.. |poe| replace:: N/A

.. |poe_option| replace:: does not support PoE

.. |lan_poe_desc| replace:: This product does not support PoE(Power over Ethernet).

.. |opt1_4g| replace:: N/A

.. |opt2_wifibt| replace:: Integrated WiFi/Bluetooth Module

.. |vin_rng| replace:: From 9V to 30V

.. |m2_spec| replace:: N/A

.. |rs232| replace:: Default to 2 x RS232 (including 1 debug port). Up to 4 x RS232 by swapping 2 x RS485 to RS232.

.. |rs485| replace:: Default to 2 x RS485. Optionally, these 2 x RS485 can be configured to RS232. Optionally, 1 x RS485 can be configured to CAN(CAN0)

.. |can| replace:: Default to 1 x CAN FD. Optionally, 1 x RS485 can be configured to CAN(CAN0), providing up to 2 x CAN FD.

.. |GPIO| replace:: N/A

.. |power_conn_desc| replace:: There is a DC input interface on this device: a **3-pin, 3.81mm screw terminal** connector.

.. |rtc| replace:: High accuracy RTC with farad capacitor, can work 1 week after power off **(default)**. High accuracy RTC with lithium coin battery, can work 3 years after power off *(optional)*.

.. |buzzer| replace:: Yes

.. |audio| replace:: 3.5mm Audio In/Out Connector, 2W Internal Speaker

.. |storage| replace:: TF Card, Supports up to 128GB SDHC

.. |os_support| replace:: Android 14, Debian12, Buildroot Linux Qt 5.15

.. |cap_screen_conn| replace:: FPC connector
